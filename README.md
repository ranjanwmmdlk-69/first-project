# Animal Husbandry Farm

A premium Next.js application for modern animal husbandry and livestock management.

## Tech Stack
- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Typography**: Outfit (Sans) & Crimson Text (Serif)
- **Deployment**: Vercel

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure
- `src/app`: Routes and layout
- `src/components`: Reusable UI components
- `src/app/globals.css`: Global styles and design system tokens

## Features
- **Livestock Categorization**: Separate modules for Poultry, Cattle, and Goats.
- **Precision Management**: Design patterns for age-based and paddock-based tracking.
- **Premium Aesthetics**: High-end farm-themed UI with smooth animations and modern typography.
